Simple Python Game

Язык: Python

Подробнее
Простой пример игры, написанной на Python с использованием pygame.

Вы играете за оранжевый кубик. Вас преследуют красные кубы. Ваша цель - как можно дольше продержаться, не задевая их. За каждую секунду Вы получаете +1 очко. Наберите как можно больше очков! Но будьте осторожнее, ведь чем больше баллов, тем больше соперников появляется на поле битвы.

Flask Task Manager

Язык: Python (Flask)

Подробнее
Простой Task Manager, написанный на Python (Flask).

В системе есть два уровня доступа:
  • администратор
  • пользователь
Администратор может добавлять новых пользователей, создавать задачи для других пользователей, подтвержать их выполнение. Каждый пользователь получает в списке только свои задачи, после выполнения которых он может сообщить об этом через веб-интерфейс.

Используется база данных.

При разработке активно использовался CSS Framework Materialize.
