Flask Task Manager

Язык: Python (Flask)

Система контроля выполнения задач пользователями со стороны администрации. В системе два уровня доступа: администратор и пользователь. Администратор может добавлять новых пользователей, создавать задачи для других пользователей, подтвержать их выполнение. Каждый пользователь получает в списке только свои задачи, после выполнения которых он может сообщить об этом через веб-интерфейс. Используется база данных. Активно применялся Materialize CSS Framework.
